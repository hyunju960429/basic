table 태그 만들기(표만들기)

html에서 표를 구현할때

  • table태그를 생성
  • tr태그생성(table row:행,줄)
  • th(table heading):행,열의 제목(굵은체,가운데정렬 되어있음)
  • td(table data):(행의 내용)열,칸
